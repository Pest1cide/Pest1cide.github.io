# Public Site Safety Checklist

Use this repository only for public profile content.

- Do not commit private source code, unpublished datasets, credentials, SSH keys, `.env` files, model checkpoints, archives, or internal lab notes.
- Link only to repositories that are already public and intended for broad viewing.
- Keep project summaries high level if the implementation is not ready to release.
- Review `git status --short` and `git diff --stat` before every commit.
- Treat every file in this repository as publishable on the public web.
