Research project · 2025–Present

Self-supervised configuration-space distance functions

A research extension of OptimalModulationDS exploring learned configuration-space distance and gradient information for obstacle-aware manipulation. The project includes distance-field training, dynamical-system and MPPI evaluations, and reinforcement-learning ablations for Franka tasks.

  • Dynamical systems

Aalto project work · Project manager · 2025

Real-time Spot digital twin

Led a team building a real-to-simulation pipeline in Genesis. The system mirrors live state from a physical Boston Dynamics Spot and maps OptiTrack human-motion data into commands for a simulated Spot, creating a practical platform for testing and interaction.

  • Genesis
  • Spot SDK
  • OptiTrack
  • Real-time systems

Control for robots with legs
